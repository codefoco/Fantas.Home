Class Animation

Animation class (Move, Scale, Rotate, Fade...)

The class Animation is used to animamate nodes. You can create a sequences of animations or a group of animations.

For instance, to make one node move from 100 points up and 100 points right you can use Animation.MoveBy

    // Create animation to move a node
    Animation moveBy = Animation.MoveBy(100, 100);

    // Run animation on node
    node.RunAnimation(moveBy);

To combine multiple animations you can use Animation.Group to execute animation in parallel or Animation.Sequence to execute in sequence.
